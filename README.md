# BarRafi EduRaport - Community Edition 🏫🚀

Selamat datang di repository resmi **BarRafi EduRaport Community Edition**. Aplikasi ini adalah solusi modern, cepat, dan offline untuk manajemen raport siswa sesuai standar **Kurikulum Merdeka**.

## 📥 Cara Download & Instalasi

Aplikasi ini bersifat **Portable** (tidak perlu instalasi atau pengaturan database yang rumit).

1. Klik tombol **[⬇️ Download BarRafi EduRaport v1.2.4](https://github.com/dayatMuhid/BarrafiEduRaport-Community/raw/main/BarRafi_EduRaport_v1.2.4_Community.zip)** untuk mengunduh versi terbaru.
2. Ekstrak file ZIP tersebut di komputer Anda.
3. Klik dua kali file **`Jalankan_EduRaport.vbs`** untuk membuka aplikasi secara instan di browser Anda.
4. Setelah selesai menggunakan aplikasi, **cukup tutup browser Anda** dan server backend akan otomatis mematikan dirinya sendiri di latar belakang secara bersih!

## 🔑 Cara Aktivasi Lisensi

Setelah membuka aplikasi, Anda akan diminta untuk melakukan aktivasi identitas:

1. Isi **Nama Lengkap** Anda pada formulir yang muncul.
2. Klik tombol **"Kirim Data ke Admin via WhatsApp"**.
3. Anda akan mendapatkan **Nomor Seri Aktivasi** dari Admin.
4. Masukkan Nomor Seri tersebut ke dalam aplikasi untuk membuka fitur penuh secara permanen.

## ✨ Fitur Utama
-   ✅ **Offline Total**: Data aman di komputer Anda sendiri tanpa koneksi internet.
-   ✅ **Penyimpanan SQLite Terenkripsi (AES-256) (Baru!)**: Data tersimpan secara persisten dan sangat aman dalam file database lokal di harddisk PC, terlindung dari pembobolan data mentah.
-   ✅ **Launcher Senyap & Auto-Shutdown (Baru!)**: Cukup klik dua kali peluncur VBS untuk membuka aplikasi tanpa terminal CMD hitam, dan cukup tutup browser untuk mematikan server secara otomatis!
-   ✅ **Import/Export Excel**: Kelola data ribuan siswa dengan mudah.
-   ✅ **Export/Import Nilai**: Export template nilai per kelas, isi di Excel, import kembali dengan deskripsi otomatis.
-   ✅ **Setup TP & CP**: Atur Tujuan Pembelajaran dan Capaian Pembelajaran sesuai BSKAP 032/2024.
-   ✅ **Template Pemerintah**: CP & TP resmi untuk Bahasa Indonesia, Matematika, IPAS, dll sudah tersedia.
-   ✅ **Auto-Generate Deskripsi**: Narasi capaian kompetensi siswa ditulis otomatis berdasarkan analisis CP & TP.
-   ✅ **Ranking Otomatis**: Posisi juara kelas terhitung secara instan.
-   ✅ **Raport Kurikulum Merdeka & P5**: Siap cetak standar nasional.
-   ✅ **One-Page PDF**: Desain raport ringkas dan hemat kertas.

## 📋 Changelog v1.2.4
- **Fitur Baru (Keamanan)**: Integrasi database SQLite dengan Enkripsi Transparan HSL AES-256-CBC untuk keamanan data rapor maksimal.

- **Fitur Baru (Auto-Shutdown)**: Sistem detak jantung browser (*heartbeat*) untuk mendeteksi penutupan browser dan mematikan server otomatis dalam 8 detik secara bersih.
