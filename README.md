# BarRafi EduRaport - Community Edition 🏫🚀

Selamat datang di repository resmi **BarRafi EduRaport Community Edition**. Aplikasi ini adalah solusi modern, cepat, dan offline untuk manajemen raport siswa sesuai standar **Kurikulum Merdeka**.

## 📥 Cara Download & Instalasi

Aplikasi ini bersifat **Portable** (tidak perlu instalasi rumit).
1.  Klik tombol **[Download ZIP](https://github.com/dayatMuhid/BarrafiEduRaport-Community/raw/main/BarRafi_EduRaport_v1.zip)** di atas.
2.  Ekstrak file ZIP tersebut di komputer Anda.
3.  Buka file `index.html` menggunakan browser (Google Chrome atau Microsoft Edge direkomendasikan).

## 🔑 Cara Aktivasi Lisensi

Setelah membuka aplikasi, Anda akan diminta untuk melakukan aktivasi identitas:
1.  Isi **Nama Lengkap** Anda pada formulir yang muncul.
2.  Klik tombol **"Kirim Data ke Admin via WhatsApp"**.
3.  Anda akan mendapatkan **Nomor Seri Aktivasi** dari Admin.
4.  Masukkan Nomor Seri tersebut ke dalam aplikasi untuk membuka fitur penuh secara permanen.

## ✨ Fitur Utama
-   ✅ **Offline Total**: Data aman di komputer Anda sendiri.
-   ✅ **Import/Export Excel**: Kelola data ribuan siswa dengan mudah.
-   ✅ **Ranking Otomatis**: Posisi juara kelas terhitung secara instan.
-   ✅ **Raport Kurikulum Merdeka & P5**: Siap cetak standar nasional.
-   ✅ **One-Page PDF**: Desain raport ringkas dan hemat kertas.
